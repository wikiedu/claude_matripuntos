# v3.0 · Premium — Spec

**Fecha:** 2026-05-02
**Versión:** v3.0
**Branch:** `feature/v3.0-premium`
**Tag al merge:** `v3.0`

## 1. Goal

Lanzar el modelo de monetización: **suscripción mensual + anual** vía Stripe, freemium B (core gratis, valor añadido pago) sin paywall en pillars de equidad. AI assistente como halo premium. App nativa React Native opcional.

## 2. Pillars de monetización

- **Free es viable**: equilibrio, tareas, calendario básico, journal básico, gamification básico — TODO funciona free para siempre.
- **Premium aporta personalización + profundidad**, no acceso a "lo importante".
- **No dark patterns**: 1-tap cancel, prorateo justo, no auto-renew agresivo.
- **Precio accesible España**: 4.99€/mes o 39€/año (proporción 35% descuento).
- **Trial 14 días**: sin tarjeta requerida (post-onboarding directo).

## 3. Tiers

### Free (siempre)
- Todo el core MVP→v2.1.
- 100 MB attachments journal.
- 3 service providers calendar.
- 90d analytics history.
- 30d journal retrospective.
- 1 referral code activo.
- Web push básico.

### Premium (4.99€/mes, 39€/año)
- 5 GB attachments.
- Service providers ilimitados.
- 365d analytics + heatmap full.
- Anuario PDF anual.
- Custom achievement badges (subir avatar custom para logros).
- Themes premium (5 paletas adicionales).
- Skip challenge semanal (regenerar el reto si no encaja).
- Multi-couple history (si tienes past-couples, acceso a 5 años).
- AI assistant (sec. 5.4).
- Priority support (24h response).
- Early access nuevas features.

## 5. Features premium específicas

### 5.1 Themes premium
- 5 paletas (blush, deep ocean, midnight, sunset, forest) además de la default.
- Cambio en tiempo real con CSS variables.
- Preview gratuito 24h.

### 5.2 AI Assistant
- **Journal summary**: pegar entry → AI sugiere temas, emociones, conexiones con entries pasadas (claude haiku, ~50 tokens cost).
- **Mood coach**: si mood "estresado" 3 días seguidos → AI sugiere prácticas (3-5 frases, no clinicidad).
- **Conflict advisor**: en negociación trabada → "¿Quieres que te sugiera una solución?". Output: 2-3 propuestas balanceadas.
- **Cost cap**: 50 interactions/user/mes. Tras eso → 0.05€/extra.
- **Privacy**: prompts sanitizados (no nombres, no detalles específicos), response no se persiste en BD del proveedor (zero-data-retention contract).

### 5.3 Multi-couple history
- Si user tiene past-couples (v1.6.1 dissolution) → acceso a stats agregadas hasta 5 años atrás.
- Free tier: solo couple activo + 1 past-couple del último año.

### 5.4 Anuario PDF
- Genera PDF visual del año (HTML→PDF via Puppeteer en backend, lazy).
- Spread por mes con highlights, fotos, frases destacadas.
- Premium-only download (con marca de agua "Powered by Matripuntos" para free preview low-res).

## 9. Out-of-scope v3.0

- Family plan (multi-couple share Premium) → v3.1
- Teen mode / minors with parental consent → v3.x lejano
- White-label B2B → no relevante
- Affiliate program → ya cubierto por referrals v2.1
- Gift subscriptions → v3.1 si demand

## 10. Riesgos

| Riesgo | Mitigación |
|---|---|
| Stripe webhooks no llegan | Polling fallback + retries |
| AI costs runaway | Hard cap por user + alertas a 80% mensual |
| Free → Premium conversion baja | A/B test del paywall + trial 14d |
| GDPR + AI provider | Anthropic ZDR contract obligatorio + DPA firmado |
| Cancelaciones masivas (refund req) | Política clara + prorateo automático |

## 11. Métricas éxito 90d post-launch

- Trial-to-paid conversion: ≥18% (industry B2C ~15-20%)
- Monthly churn: ≤8%
- ARR target trimestre: 5k€ (1000 paying x 5€)
- AI usage Premium: ≥40% (señal de valor)
- Free retention D30: ≥35% (no se va por paywall agresivo)

## 12. Compliance

- Stripe Atlas / DPA firmado.
- Anthropic ZDR + DPA.
- IVA España: facturación automática vía Stripe Tax.
- Política de reembolso: 7 días sin preguntas (post-trial).
